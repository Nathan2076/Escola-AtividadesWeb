Substituir uma String em um Texto


var originalText = document.getElementById("original-text").innerHTML;

document.getElementById("modified-text").innerHTML = originalText.replace("dolor", "<em>color</em>");

O código possui uma variável originalText na primeira linha, que guarda o valor do texto base (original).

O código presente na terceira linha modifica o HTML interno do elemento com o ID modified-text que, até então, está vazio, e aplica o texto original com as modificações, feitas através da função nativa {variavel}.replace(), que leva dois argumentos: a expressão a ser encontrada no texto original e a expressão a substituir a original, respectivamente.

Por fim, ambos os textos são mostrados ao usuário, possibilitando a verificação das diferenças entre os mesmos. A modificação aplicada recebe ênfase com a tag <em></em>.
