Encontrar uma String em um Texto


var str = "Lorem ipsum dolor sit amet, consectetur adipiscing elit.";

if (str.match(/amet/))
{
    document.getElementById("status").innerHTML += "Expressão Encontrada.";
}
else
{
    document.getElementById("status").innerHTML += "Expressão Não Encontrada.";
}

O código possui uma variável str na primeira linha, que guarda o valor do texto base.

A condição presente na terceira linha verifica se a expressão dentro de {variavel}.match(), uma função nativa do JavaScript, está presente no texto base.
Obs.: a expressão a ser verificada deve obrigatoriamente estar entre //, e a verificação é case sensitive.

Caso a verificação seja concluída com êxito, isto é, caso a expressão fornecida exista no texto base, o código adiciona "Expressão Encontrada." ao status. Caso contrário, a mensagem "Expressão Não Encontrada." aparece no lugar.
